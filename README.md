# Webpack / Babel Workflow


**Version**

**1.0.0**


# Usage

**Installation**

Install the dependancies

```$ npm install```

**Dev Server**

To run the dev server - Runs on localhost: 8080

```$ npm start```

**Dev mode build**

Builds a dev version to run in the server

```$ npm run dev```

**Build**

Builds a production build and complies all the HTML, JS, CSS

```$ npm run build```
